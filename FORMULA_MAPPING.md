# สูตรหลักในแต่ละหมวด

| เครื่องมือในเว็บ | สูตรหลัก |
|---|---|
| ค่ายกโค้งและความเร็ว | `Ce=8.416V²/R`, `Ca=5.611(V+5)²/R`, `Cd=Ce−Ca` |
| ความเร็วผ่านประแจ | `ac=g·hd/G`, `V=3.6√(acR)` |
| ออกแบบโครงสร้างทางรถไฟและระบายน้ำ | คันทาง: `qu=c·Nc+γ·Df·Nq+0.5·γ·B·Nγ`, `qa=qu/FS` · ระบายน้ำ: Rational `Q=0.278CIA` + Manning `Q=(1/n)AR^(2/3)S^(1/2)` |
| ราง–หมอน–หินโรยทาง | Beam on elastic foundation และการกระจายแรงผ่านชั้นทาง |
| เรขาคณิตประแจ | `θ=2atan(0.5/N)` และองค์ประกอบเรขาคณิตพื้นฐาน |
| โค้งต่อและโค้งกลม | `Ls=0.01VCa`, `Shift=Ls²/(24R)` |
| การขยายขนาดทาง | รัศมีจาก cant deficiency และระยะทางตรง `0.2V` |

ค่าคงที่ เกณฑ์ตรวจสอบ (มบท.) และคุณสมบัติรางถูกเก็บใน `calculators.py` ส่วนภาพประกอบทางวิศวกรรมอยู่ใน `diagrams.py` เพื่อให้ตรวจสอบและแก้ไขได้จากจุดเดียว
