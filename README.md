# การเชื่อมโยงสูตรจาก Excel

| เครื่องมือในเว็บ | ชีตอ้างอิง | สูตรหลัก |
|---|---|---|
| ค่ายกโค้งและความเร็ว | ความเร็วโค้ง, Cant Calc | `Ce=8.416V²/R`, `Ca=5.611(V+5)²/R` |
| ความเร็วผ่านประแจ | ความเร็วผ่านประแจ | `ac=g·hd/G`, `V=3.6√(acR)` |
| ทางระบายน้ำ | Drainage Design | Rational Formula และ Manning Formula |
| ราง–หมอน–หินโรยทาง | หมอนคอนกรีต, ข้อมูลINPUT | Beam on elastic foundation และการกระจายแรงผ่านชั้นทาง |
| เรขาคณิตประแจ | Turnout Design, Turnout Design สรุป | `θ=2atan(0.5/N)` และองค์ประกอบเรขาคณิตพื้นฐาน |
| โค้งต่อและโค้งกลม | คำนวณโค้งต่อ+กลม | `Ls=0.01VCa`, `Shift=Ls²/(24R)` |
| โค้งเส้นเชือก | String Lining ทส.1, String Lining ทส.2, คู่มือหน้า 33-36 | `D=Vเดิม-Vใหม่`, `Eᵢ=Eᵢ₋₁+Dᵢ`, `Fᵢ=Fᵢ₋₁+Eᵢ₋₁`, `ดัด=2F` |
| การขยายขนาดทาง | มุมประแจ, การขยายขนาดทาง จากราง | รัศมีจาก cant deficiency และระยะทางตรง `0.2V` |

ค่าคงที่ เกณฑ์ตรวจสอบ และรายการคุณสมบัติรางถูกเก็บใน `calculators.py` เพื่อให้ตรวจสอบและแก้ไขได้จากจุดเดียว

ปุ่มปรับปรุง Vใหม่ใช้การแก้ระบบสมการแบบ least-squares โดยบังคับ `ΣVเดิม=ΣVใหม่`, ผลรวมโมเมนต์ของผลต่างเป็นศูนย์, Vต้นและ Vปลายเป็นศูนย์ และผสมค่ากลับเข้าหา Vเดิมเมื่อจำเป็นเพื่อลดระยะดัดให้อยู่ในเป้าหมาย 60-120 มม.
